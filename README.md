# QA-Testing-Automation-Playwright

OrangeHRM Automation using Playwright (Node.js)

End-to-end UI automation framework built with Playwright and Node.js for the OrangeHRM demo application.
This project covers login validation, dashboard verification, PIM employee creation flow, and My Info navigation, with a focus on stable locators, explicit waits, and human-readable execution speed.

Key highlights:

Playwright Test Runner with Node.js

Cross-browser testing (Chromium, Firefox, WebKit)

Robust waits for SPA applications

Screenshot, video, and trace capture on failures

Configured for both local execution and CI pipelines
